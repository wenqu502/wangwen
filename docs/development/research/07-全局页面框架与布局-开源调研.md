## 一、调研背景

织文 WangWen 采用「画布 + AI 对话面板」的双栏布局，需要在有限的桌面空间内同时展示可视化内容和 AI 交互界面。参考目标：

- Figma：左侧工具栏 + 中央画布 + 右侧属性面板
- Notion：左侧导航 + 中央内容 + 右侧评论
- VS Code：左侧活动栏 + 中央编辑器 + 右侧面板

---

## 二、设计决策清单

### 2.1 整体布局

```
┌─────────────────────────────────────────────────────────────┐
│  SiteHeader (52px)                                          │
├────────┬──────────────────────────────┬─────────────────────┤
│        │                              │                     │
│ LeftBar│  Canvas Area                 │ AI Panel (320px)    │
│ 60px   │  - 角色卡片墙 / 剧情分支树    │ - AI 创作助手        │
│ 可折叠  │  - 人物关系图 / 体系结构      │ - 对话历史           │
│        │                              │ - 输入框             │
│        │                              │                     │
└────────┴──────────────────────────────┴─────────────────────┘
```

### 2.2 各区域规范

| 区域 | 宽度 | 说明 |
|------|------|------|
| SiteHeader | 100% | 固定 52px |
| LeftBar | 60px（展开 200px） | 可折叠，工具切换 |
| Canvas Area | 自适应 | flex: 1，overflow: auto |
| AI Panel | 320px | 固定宽度，可收起 |

### 2.3 交互细节

| 决策项 | 方案 |
|--------|------|
| LeftBar 折叠 | 点击折叠按钮，收起为 60px 图标栏 |
| AI Panel 收起 | 点击边缘按钮，收起为右侧 8px 细条 |
| 作品名截断 | max-width: 160px，ellipsis 截断 |
| Tab 切换 | 顶部 Tab 切换角色/剧情/关系/体系/灵感 |
| 画布 Tab | Canvas 内部二级 Tab 切换不同视图 |
