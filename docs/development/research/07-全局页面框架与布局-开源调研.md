## 一、需求背景与范围界定

### 1.1 PRD 原文需求摘录

根据《PRD-网文创作辅助工具-v3.0.md》：

> **主界面布局**：
> - 顶部导航栏（52px）：作品选择 | 全局搜索 | 导出 | 设置
> - 左侧：主内容区域（可视化画布 / 卡片列表）
> - 右侧：AI 对话面板（常驻）
>
> **核心交互范式**：
> - 左侧可视化画布：角色卡片墙 / 剧情分支树 / 关系网络图 / 体系结构图
> - 右侧 AI 面板：AI 创作助手 / 对话历史 / 输入框
> - 底部：模块切换 Tab（角色 / 剧情 / 关系 / 体系 / 灵感）
>
> **页面跳转链路**：
> - 角色小像 → 点击「首次登场」→ 剧情分支树对应节点
> - 角色小像 → 点击「关联角色」→ 人物关系图并高亮目标
> - 剧情分支树 → 点击节点中的角色名 → 角色小像卡片侧边栏
> - 人物关系图 → 点击角色节点 → 角色小像侧边栏

根据《PRD-网文创作辅助工具-v2.0.md》补充：

> - 左侧功能导航（215px）：角色 / 剧情 / 关系 / 体系 / 灵感 Tab 切换
> - 模块主内容区域：自适应宽度
> - 最小宽度：1380px

### 3.1 Tailwind CSS + 自定义组件 —— 原子化 CSS 方案

| 项 | 内容 |
|----|------|
| 官网 | https://tailwindcss.com |
| GitHub | https://github.com/tailwindlabs/tailwindcss |
| Stars | 82k+ |
| 最新版本 | v4.x |
| 技术栈 | PostCSS + CSS |
| 开源协议 | MIT |
| 定位 | 原子化 CSS 框架 |
| 维护团队 | Tailwind Labs |
| 更新频率 | 非常活跃 |

**核心能力详解**

1. **原子化 CSS**：
   - 大量工具类（utility classes）
   - 无需写自定义 CSS
   - 响应式前缀（sm: / md: / lg:）

2. **JIT 编译器**：
   - 只生成使用到的样式
   - 包体积极小（生产环境 ~10KB）

3. **自定义主题**：
   - `tailwind.config.js` 配置设计 Token
   - 颜色、间距、字体等全部可定制

4. **Dark Mode**：
   - 内置暗色模式支持
   - `dark:` 前缀切换样式

**与织文需求的匹配度分析**

| 织文需求 | Tailwind CSS 支持情况 | 实现复杂度 |
|----------|----------------------|------------|
| 三栏布局 | flex / grid 工具类 | 低 |
| 面板折叠 | w-0 / w-60 切换 | 低 |
| 固定高度 | h-12 / h-screen | 低 |
| 响应式 | 前缀工具类 | 低 |
| 暗色模式 | dark: 前缀 | 低 |
| 自定义主题 | tailwind.config.js | 低 |

**优缺点**

| 优点 | 缺点 |
|------|------|
| 开发效率高 | 类名冗长 |
| 包体积极小 | 学习成本 |
| 设计系统友好 | 无内置组件 |
| 暗色模式内置 | — |

**对织文的借鉴点**

| 借鉴项 | 具体应用 |
|--------|----------|
| flex 布局 | SiteHeader + LeftBar + Canvas + AI Panel 三栏布局 |
| 工具类 | 所有间距、颜色、字体统一使用工具类 |
| 暗色模式 | dark: 前缀实现主题切换 |
| JIT 编译 | 只生成使用的样式，包体积极小 |

## 四、综合评分与选型决策

### 4.1 评分矩阵

| 维度（权重） | Tailwind CSS | Radix UI | shadcn/ui | react-resizable-panels | Lucide React |
|-------------|--------------|----------|-----------|------------------------|--------------|
| 技术栈匹配度（25%） | 5 | 5 | 5 | 5 | 5 |
| 布局灵活性（25%） | 5 | 2 | 2 | 4 | 1 |
| 组件生态（20%） | 1 | 4 | 5 | 1 | 2 |
| 性能表现（15%） | 5 | 5 | 4 | 5 | 5 |
| 社区活跃度（10%） | 5 | 4 | 5 | 3 | 4 |
| 许可证（5%） | 5 | 5 | 5 | 5 | 5 |
| **加权总分** | **4.15** | **3.65** | **3.85** | **3.55** | **3.05** |

**评分细则**

- **Tailwind CSS（4.15/5）**：
  - 布局灵活性最强
  - 包体积极小
  - 但无内置组件

- **shadcn/ui（3.85/5）**：
  - 组件质量高，完全可定制
  - 但布局能力依赖 Tailwind

- **Radix UI（3.65/5）**：
  - 无头原语，可访问性优秀
  - 但需自行实现样式

- **react-resizable-panels（3.55/5）**：
  - 面板调整能力强
  - 但不支持折叠

- **Lucide React（3.05/5）**：
  - 图标库，功能单一

### 4.2 选型结论

**主选方案组合**：

1. **Tailwind CSS**：布局基础，所有间距/颜色/字体统一使用工具类
2. **shadcn/ui**：组件基础，Button / Card / Dialog / Tabs 等组件
3. **Radix UI**：底层原语，Tooltip / DropdownMenu / Accordion 等
4. **Lucide React**：图标库，所有 UI 图标

**辅助方案**：

- **react-resizable-panels**：若需要用户自定义面板宽度时使用

## 六、Zustand Store 设计

```typescript
// src/layout/store.ts

import { create } from 'zustand'
import { immer } from 'zustand/middleware/immer'
import { persist } from 'zustand/middleware'
import { AppModule, LayoutState, Work } from './types'

interface LayoutStore extends LayoutState {
  // 模块切换
  setActiveModule: (module: AppModule) => void

  // 面板折叠
  toggleLeftBar: () => void
  toggleAiPanel: () => void

  // 面板宽度
  setLeftBarWidth: (width: number) => void
  setAiPanelWidth: (width: number) => void

  // 主题
  toggleDarkMode: () => void

  // 作品
  currentWork: Work | null
  setCurrentWork: (work: Work | null) => void
}

export const useLayoutStore = create<LayoutStore>()(
  immer(
    persist(
      (set) => ({
        // 初始状态
        activeModule: AppModule.CHARACTER,
        leftBarCollapsed: false,
        aiPanelCollapsed: false,
        leftBarWidth: 200,
        aiPanelWidth: 320,
        isDarkMode: false,
        currentWork: null,

        // 模块切换
        setActiveModule: (module) =>
          set((state) => {
            state.activeModule = module
          }),

        // 面板折叠
        toggleLeftBar: () =>
          set((state) => {
            state.leftBarCollapsed = !state.leftBarCollapsed
          }),

        toggleAiPanel: () =>
          set((state) => {
            state.aiPanelCollapsed = !state.aiPanelCollapsed
          }),

        // 面板宽度
        setLeftBarWidth: (width) =>
          set((state) => {
            state.leftBarWidth = width
          }),

        setAiPanelWidth: (width) =>
          set((state) => {
            state.aiPanelWidth = width
          }),

        // 主题
        toggleDarkMode: () =>
          set((state) => {
            state.isDarkMode = !state.isDarkMode
          }),

        // 作品
        setCurrentWork: (work) =>
          set((state) => {
            state.currentWork = work
          }),
      }),
      {
        name: 'wangwen-layout',
        partialize: (state) => ({
          leftBarCollapsed: state.leftBarCollapsed,
          aiPanelCollapsed: state.aiPanelCollapsed,
          leftBarWidth: state.leftBarWidth,
          aiPanelWidth: state.aiPanelWidth,
          isDarkMode: state.isDarkMode,
        }),
      }
    )
  )
)
```

        </button>
      ))}
    </aside>
  )
}
