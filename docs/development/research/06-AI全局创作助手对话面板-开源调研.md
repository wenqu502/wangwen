## 一、调研背景

AI 全局创作助手是织文 WangWen 的核心交互入口，常驻在右侧边栏。用户通过自然语言与 AI 对话，完成角色创建、剧情规划、关系梳理、体系设计等创作任务。核心需求：

- 多轮对话上下文记忆
- AI 消息展示（文本/建议/操作按钮）
- 用户输入框（支持快捷指令）
- 对话历史浏览
- 流式响应（打字机效果）

---

## 二、候选项目分析

### 2.1 对话 UI 组件库

| 项目 | 说明 | 推荐度 |
|------|------|--------|
| **react-chatbot-kit** | React 聊天机器人 UI 套件，支持自定义消息类型 | ★★★ |
| **chat-ui-kit-react** | 轻量级聊天 UI 组件 | ★★ |
| **@chatscope/chat-ui-kit-react** | 功能全面的聊天 UI 套件 | ★★★★ |
| **自研** | 使用 Tailwind 自定义，灵活性最高 | ★★★★★ |

**推荐**：自研为主，参考 @chatscope/chat-ui-kit-react 的布局模式

### 2.2 流式响应

| 方案 | 说明 |
|------|------|
| SSE (Server-Sent Events) | 标准方案，兼容性好 |
| WebSocket | 双向通信，适合复杂场景 |
| Fetch ReadableStream | 现代浏览器原生支持 |

**推荐**：SSE 或 Fetch ReadableStream（取决于后端 AI 接口支持）

## 三、设计决策清单

| 决策项 | 方案 |
|--------|------|
| 面板宽度 | 320px，固定宽度 |
| 消息气泡 | AI 白底左对齐，用户蓝底右对齐 |
| 头像 | AI 用机器人图标，用户用默认头像 |
| 时间戳 | 11px，#86909c，每条消息下方 |
| 输入框 | 底部固定，多行文本，支持 Enter 发送 / Shift+Enter 换行 |
| 快捷指令 | 输入框上方快捷按钮（生成角色/续写剧情/梳理关系） |
| 加载状态 | AI 输入指示器（三颗跳动圆点） |
| 错误处理 | 发送失败显示重试按钮 |
| 历史记录 | 滚动加载更多历史对话 |
