## 一、调研背景

织文 WangWen 的「角色小像」模块是用户与 AI 协作创建、管理和展示角色档案的核心功能。用户通过自然语言描述，AI 生成完整角色档案（姓名、外貌、性格、背景、创伤、目标等），以卡片形式展示在画布上。核心需求：

- AI 对话式创建/修改角色
- AI 形象生成（文生图）
- 角色卡片墙展示（网格/列表布局）
- 人设一致性校验结果展示
- 角色详情查看与编辑

## 二、候选项目分析

### 2.1 角色卡片设计 —— 无特定开源库，参考通用设计模式

角色卡片是通用 UI 组件，无需专用库，主要参考：

1. **Pinterest / Masonry 网格**：卡片墙瀑布流布局
2. **Trello / 看板**：卡片拖拽排序
3. **Notion 数据库视图**：卡片/表格/列表多视图切换

**设计模式总结**

| 模式 | 特征 | 适用场景 |
|------|------|----------|
| 瀑布流网格 | 不等高卡片，自动填充空白 | 角色卡片墙（视觉丰富） |
| 等高网格 | 等高卡片，整齐排列 | 角色卡片墙（信息密度高） |
| 列表视图 | 横向卡片，一行一个角色 | 角色列表（信息全面） |
| 看板视图 | 按阵营/身份分组拖拽 | 角色分组管理 |

### 2.2 文生图集成 —— 角色形象生成

| 方案 | 说明 | 开源/商用 |
|------|------|-----------|
| Stable Diffusion API | 本地/远程部署 SD 模型 | 开源 |
| Midjourney API | 商业 API，质量高 | 商用 |
| DALL-E 3 API | OpenAI 商业 API | 商用 |
| Pollinations.ai | 免费开源文生图 API | 开源 |
| SiliconFlow/硅基流动 | 国内 API 平台，支持 FLUX | 商用 |

**推荐方案**：

- **开发期**：使用 Pollinations.ai（免费，无需 Key）或 SiliconFlow（国内稳定）
- **生产期**：接入 SiliconFlow 或自建 Stable Diffusion

### 2.3 富文本编辑 —— 角色档案编辑

| 项目 | 技术栈 | 特点 |
|------|--------|------|
| TipTap | ProseMirror + Vue/React | 现代、可扩展、协作友好 |
| Quill | 原生 JS | 轻量、成熟、主题丰富 |
| Slate.js | React | 完全可定制、数据驱动 |
| CKEditor 5 | 原生 JS | 功能全面、商业支持 |

**推荐**：TipTap（React 生态，与织文技术栈匹配，可扩展性强）

## 三、设计决策清单

### 3.1 角色卡片

| 决策项 | 方案 |
|--------|------|
| 卡片尺寸 | 160px（缩略）/ 200px（标准）/ 280px（详情） |
| 布局方式 | 等高网格为主（flex-wrap），支持切换列表视图 |
| 头像 | 渐变圆形，48px，姓氏首字母 |
| 名字 | 15px，600 字重，居中 |
| 简介 | 12px，2 行截断，#86909c |
| 标签 | 底部语义 Tag（男主/女主/反派/配角） |
| hover | 阴影加深 + 轻微上浮 translateY(-2px) |
| 点击 | 选中态（蓝色边框）或打开详情 Drawer |

### 3.2 角色卡片墙

| 决策项 | 方案 |
|--------|------|
| 默认视图 | 网格视图（3-4 列） |
| 切换视图 | 网格 / 列表 / 看板（按阵营分组） |
| 筛选 | 按身份/阵营/性别/状态筛选 |
| 搜索 | 实时搜索角色名和标签 |
| 排序 | 按创建时间 / 出场顺序 / 名字排序 |
| 空状态 | 占位插画 + 「让 AI 帮你创建第一个角色」 |
| 批量操作 | 框选多角色，批量删除/导出 |

### 3.3 角色详情 Drawer

| 决策项 | 方案 |
|--------|------|
| 打开方式 | 点击卡片从右侧滑出 Drawer |
| 面板宽度 | 400px |
| 内容结构 | 头像区 → 基础信息 → 性格档案 → 背景故事 → 关系 → 台词 |
| 编辑模式 | 点击字段进入编辑，保存后更新 |
| AI 建议 | 底部悬浮「AI 建议」按钮，点击展开修改建议 |

### 3.4 AI 形象生成

| 决策项 | 方案 |
|--------|------|
| 触发方式 | 角色详情页点击「生成形象」 |
| 生成过程 | 显示 Loading 状态 + 进度提示 |
| 结果展示 | 4 张备选图网格，用户选择保存 |
| 风格调整 | 支持输入风格描述（如「更冷峻一些」） |
| 失败处理 | 显示重试按钮 + 错误提示 |
