## 一、需求背景与范围界定

### 1.1 PRD 原文需求摘录

根据《PRD-网文创作辅助工具-v3.0.md》第 5.1 节「角色小像」：

> **AI 对话式创建**：用户用一句话描述角色，AI 生成完整档案
> **AI 对话式修改**：支持"把他的性格改得更极端一点"等上下文指令
> **人设一致性校验**：修改角色档案后自动扫描已写章节，标记冲突
> **角色档案数据结构**：包含姓名、外貌、性格、背景、创伤、目标、弧光、台词等

## 三、全文搜索方案

### 3.1 候选方案

#### 方案 A：Dexie + FlexSearch（推荐）

```typescript
import FlexSearch from 'flexsearch'

class CharacterSearch {
  private index = new FlexSearch.Index({ tokenize: 'forward' })
  
  async rebuild(workId: string) {
    const chars = await db.characters.where('workId').equals(workId).toArray()
    this.index.clear()
    chars.forEach(c => {
      this.index.add(c.id, [
        c.name,
        ...c.aliases,
        ...c.tags,
        c.appearance,
        c.personality.surface,
        c.personality.inner,
        c.background,
      ].join(' '))
    })
  }
  
  search(query: string): string[] {
    return this.index.search(query) as string[]
  }
}
```

**优势**：轻量，中文支持好，增量更新
**劣势**：内存占用，需手动同步

#### 方案 B：Dexie 原生 `startsWith`

```typescript
// 仅支持前缀搜索
const results = await db.characters
  .where('name')
  .startsWithIgnoreCase(query)
  .toArray()
```

**优势**：零依赖，最简单
**劣势**：仅前缀匹配，不支持内容搜索

#### 方案 C：Fuse.js（模糊搜索）

```typescript
import Fuse from 'fuse.js'

const fuse = new Fuse(characters, {
  keys: ['name', 'aliases', 'tags', 'appearance'],
  threshold: 0.4,
})

const results = fuse.search(query)
```

**优势**：模糊匹配，容错性好
**劣势**：大数据量时内存占用高

### 3.2 推荐方案

**FlexSearch（方案 A）**。原因：
- 网文创作中角色名搜索是高频场景
- FlexSearch 支持中文分词（forward tokenize）
- 增量更新机制适合实时编辑场景

### 4.2 校验流程

```typescript
async function checkCharacterConsistency(workId: string, characterId: string) {
  const character = await db.characters.get(characterId)
  const plotNodes = await db.plotNodes.where('workId').equals(workId).toArray()
  
  const conflicts: Conflict[] = []
  
  for (const node of plotNodes) {
    // AI 分析节点内容中是否存在与角色设定冲突的描述
    const analysis = await aiAnalyzeConflict(character, node.summary)
    if (analysis.hasConflict) {
      conflicts.push({
        nodeId: node.id,
        nodeTitle: node.title,
        conflictType: analysis.conflictType,
        description: analysis.description,
      })
    }
  }
  
  // 保存校验结果
  await db.consistencyChecks.put({
    id: generateId(),
    workId,
    characterId,
    conflicts,
    checkedAt: Date.now(),
  })
  
  return conflicts
}
```

## 五、Zustand Store 设计

```typescript
// src/modules/character/store.ts

import { create } from 'zustand'
import { immer } from 'zustand/middleware/immer'
import { db } from '@/db'

interface CharacterState {
  characters: Character[]
  selectedId: string | null
  filter: CharacterFilter
  searchQuery: string
  isLoading: boolean
  
  // Actions
  loadCharacters: (workId: string) => Promise<void>
  createCharacter: (workId: string, data: CharacterFormData) => Promise<string>
  updateCharacter: (id: string, data: Partial Partial<Character>) => Promise<void>
  deleteCharacter: (id: string) => Promise<void>
  selectCharacter: (id: string | null) => void
  setFilter: (filter: CharacterFilter) => void
  setSearchQuery: (query: string) => void
}

export const useCharacterStore = create create<CharacterState>()(
  immer((set, get) => ({
    characters: [],
    selectedId: null,
    filter: {},
    searchQuery: '',
    isLoading: false,
    
    async loadCharacters(workId) {
      set({ isLoading: true })
      const chars = await db.characters.where('workId').equals(workId).toArray()
      set({ characters: chars, isLoading: false })
    },
    
    async createCharacter(workId, data) {
      const character: Character = {
        id: generateId(),
        workId,
        ...data,
        createdAt: Date.now(),
        updatedAt: Date.now(),
      }
      await db.characters.add(character)
      set(state => { state.characters.push(character) })
      return character.id
    },
    
    async updateCharacter(id, data) {
      await db.characters.update(id, { ...data, updatedAt: Date.now() })
      set(state => {
        const idx = state.characters.findIndex(c => c.id === id)
        if (idx !== -1) {
          state.characters[idx] = { ...state.characters[idx], ...data, updatedAt: Date.now() }
        }
      })
    },
    
    async deleteCharacter(id) {
      await db.characters.delete(id)
      set(state => {
        state.characters = state.characters.filter(c => c.id !== id)
      })
    },
    
    selectCharacter(id) {
      set({ selectedId: id })
    },
    
    setFilter(filter) {
      set({ filter })
    },
    
    setSearchQuery(query) {
      set({ searchQuery: query })
    },
  }))
)
```
