## 一、需求背景与范围界定

### 1.1 PRD 原文需求摘录

根据《PRD-网文创作辅助工具-v3.0.md》第 5.5 节「灵感便签」：

> **核心理念**：AI 自动从对话中提取灵感，用户随时调取。
>
> **AI 自动提取**：
> - 用户在对话中随口提到的好点子，AI 自动检测并保存
> - 自动编号（如 #I-047）
> - 自动推测标签（如 [角色] [情感冲击] [剧情转折]）
> - 提供关联选项：关联角色档案 / 剧情树节点 / 仅保存
>
> **AI 灵感激活**：
> - 用户卡文时，AI 主动推荐未使用的灵感
> - 按相关性排序（与当前写作位置关联度）
> - 支持自然语言操作（如「把第1条插入到下一章」）

根据《PRD-网文创作辅助工具-v2.0.md》补充界面需求：

> - 灵感便签：快速记录 + 列表 + 拖拽关联到节点/角色
> - 五大分类可正常录入

| 项 | 内容 |

**核心能力详解**

   - `@dnd-kit/sortable`：排序列表/网格

   - `useSortable` Hook
   - `SortableContext` 组件
   - 支持列表和网格排序
   - 支持多容器拖拽（跨列拖拽）

4. **可访问性**：
   - 完整的键盘操作支持
   - ARIA 属性自动管理

## 四、综合评分与选型决策

### 4.1 评分矩阵

| 维度（权重） | @dnd-kit | react-beautiful-dnd | React-Kanban | 自定义方案 | react-masonry-css |
|-------------|----------|---------------------|--------------|-----------|-------------------|
| 技术栈匹配度（25%） | 5 | 4 | 4 | 5 | 5 |
| 拖拽能力（25%） | 4 | 4 | 3 | 5 | 1 |
| 自定义能力（20%） | 5 | 3 | 3 | 5 | 3 |
| 性能表现（15%） | 5 | 4 | 4 | 4 | 4 |
| 社区活跃度（10%） | 4 | 3 | 2 | 3 | 3 |
| 许可证（5%） | 5 | 5 | 5 | 5 | 5 |
| **加权总分** | **4.55** | **3.80** | **3.40** | **4.65** | **3.35** |

**评分细则**

- **自定义方案（4.65/5）**：
  - 完全自由，可实现看板 + 自由画布双模式
  - 但开发工作量大

- **@dnd-kit（4.55/5）**：
  - 现代 React API
  - 看板模式需自行组装
  - 自由画布需额外开发

- **react-beautiful-dnd（3.80/5）**：
  - 看板模式成熟
  - 但维护模式

- **React-Kanban（3.40/5）**：
  - 社区小，维护慢

- **react-masonry-css（3.35/5）**：
  - 仅布局，无拖拽

### 4.2 选型结论

**主选方案：@dnd-kit + 自定义画布**

理由：
1. **看板模式**：@dnd-kit 的 `SortableContext` + 多容器实现看板列拖拽
2. **自由画布**：自定义 Pointer Events 实现便签自由摆放
3. **现代 React**：Hooks API，TypeScript 原生支持
4. **轻量**：核心 ~10KB gzip
5. **可访问性**：内置键盘操作支持

**辅助方案：react-masonry-css**

若需要瀑布流布局的便签墙视图时使用。

## 六、Zustand Store 设计

```typescript
// src/modules/idea-stash/store.ts

import { create } from 'zustand'
import { immer } from 'zustand/middleware/immer'
import { persist } from 'zustand/middleware'
import { IdeaNote, NoteCategory, NoteStatus, NoteLink, IdeaStashState } from './types'

interface IdeaStashStore extends IdeaStashState {
  // CRUD
  addNote: (note: IdeaNote) => void
  updateNote: (id: string, updates: Partial<IdeaNote>) => void
  removeNote: (id: string) => void

  // 状态变更
  archiveNote: (id: string) => void
  useNote: (id: string) => void
  moveNoteCategory: (id: string, category: NoteCategory) => void

  // 关联
  addLink: (noteId: string, link: NoteLink) => void
  removeLink: (noteId: string, linkId: string) => void

  // 筛选
  setFilterCategory: (category: NoteCategory | null) => void
  setFilterStatus: (status: NoteStatus | null) => void
  setSearchQuery: (query: string) => void
  setViewMode: (mode: 'kanban' | 'list' | 'canvas') => void

  // 选中
  selectNote: (id: string | null) => void

  // 画布位置
  updateNotePosition: (id: string, x: number, y: number) => void

  // 计算属性
  getFilteredNotes: () => IdeaNote[]
  getNotesByCategory: (category: NoteCategory) => IdeaNote[]
  getKanbanColumns: () => { id: NoteCategory; title: string; notes: IdeaNote[] }[]
}

export const useIdeaStashStore = create<IdeaStashStore>()(
  immer(
    persist(
      (set, get) => ({
        // 初始状态
        notes: [],
        filterCategory: null,
        filterStatus: null,
        searchQuery: '',
        viewMode: 'kanban',
        selectedNoteId: null,

        // CRUD
        addNote: (note) =>
          set((state) => {
            state.notes.push(note)
          }),

        updateNote: (id, updates) =>
          set((state) => {
            const note = state.notes.find((n) => n.id === id)
            if (note) {
              Object.assign(note, updates, { updatedAt: new Date().toISOString() })
            }
          }),

        removeNote: (id) =>
          set((state) => {
            state.notes = state.notes.filter((n) => n.id !== id)
          }),

        // 状态变更
        archiveNote: (id) =>
          set((state) => {
            const note = state.notes.find((n) => n.id === id)
            if (note) {
              note.status = NoteStatus.ARCHIVED
            }
          }),

        useNote: (id) =>
          set((state) => {
            const note = state.notes.find((n) => n.id === id)
            if (note) {
              note.status = NoteStatus.USED
            }
          }),

        moveNoteCategory: (id, category) =>
          set((state) => {
            const note = state.notes.find((n) => n.id === id)
            if (note) {
              note.category = category
            }
          }),

        // 关联
        addLink: (noteId, link) =>
          set((state) => {
            const note = state.notes.find((n) => n.id === noteId)
            if (note) {
              note.links.push(link)
            }
          }),

        removeLink: (noteId, linkId) =>
          set((state) => {
            const note = state.notes.find((n) => n.id === noteId)
            if (note) {
              note.links = note.links.filter((l) => l.id !== linkId)
            }
          }),

        // 筛选
        setFilterCategory: (category) =>
          set((state) => {
            state.filterCategory = category
          }),

        setFilterStatus: (status) =>
          set((state) => {
            state.filterStatus = status
          }),

        setSearchQuery: (query) =>
          set((state) => {
            state.searchQuery = query
          }),

        setViewMode: (mode) =>
          set((state) => {
            state.viewMode = mode
          }),

        // 选中
        selectNote: (id) =>
          set((state) => {
            state.selectedNoteId = id
          }),

        // 画布位置
        updateNotePosition: (id, x, y) =>
          set((state) => {
            const note = state.notes.find((n) => n.id === id)
            if (note) {
              note.x = x
              note.y = y
            }
          }),

        // 计算属性
        getFilteredNotes: () => {
          const { notes, filterCategory, filterStatus, searchQuery } = get()
          return notes.filter((note) => {
            if (filterCategory && note.category !== filterCategory) return false
            if (filterStatus && note.status !== filterStatus) return false
            if (searchQuery) {
              const query = searchQuery.toLowerCase()
              return (
                note.content.toLowerCase().includes(query) ||
                note.tags.some((t) => t.toLowerCase().includes(query))
              )
            }
            return true
          })
        },

        getNotesByCategory: (category) => {
          return get().notes.filter((n) => n.category === category)
        },

        getKanbanColumns: () => {
          const filtered = get().getFilteredNotes()
          return [
            { id: NoteCategory.IDEA, title: '灵感', notes: filtered.filter((n) => n.category === NoteCategory.IDEA) },
            { id: NoteCategory.FORESHADOW, title: '伏笔', notes: filtered.filter((n) => n.category === NoteCategory.FORESHADOW) },
            { id: NoteCategory.RULE, title: '规则', notes: filtered.filter((n) => n.category === NoteCategory.RULE) },
            { id: NoteCategory.TODO, title: '待办', notes: filtered.filter((n) => n.category === NoteCategory.TODO) },
          ]
        },
      }),
      {
        name: 'wangwen-idea-stash',
        partialize: (state) => ({
          notes: state.notes,
          filterCategory: state.filterCategory,
          filterStatus: state.filterStatus,
          viewMode: state.viewMode,
        }),
      }
```

### 7.2 看板视图

```tsx
// src/modules/idea-stash/components/KanbanBoard.tsx

import React from 'react'
import { DndContext, DragEndEvent, MouseSensor, TouchSensor, useSensor, useSensors } from '@dnd-kit/core'
import { SortableContext, verticalListSortingStrategy } from '@dnd-kit/sortable'
import { IdeaNoteCard } from './IdeaNoteCard'
import { useIdeaStashStore } from '../store'
import { NoteCategory } from '../types'

export const KanbanBoard: React.FC = () => {
  const { getKanbanColumns, moveNoteCategory } = useIdeaStashStore()
  const columns = getKanbanColumns()

  const sensors = useSensors(
    useSensor(MouseSensor, { activationConstraint: { distance: 5 } }),
    useSensor(TouchSensor, { activationConstraint: { distance: 5 } })
  )

  const handleDragEnd = (event: DragEndEvent) => {
    const { active, over } = event
    if (!over) return

    const noteId = active.id as string
    const overId = over.id as string

    // 判断是放到列上还是卡片上
    if (Object.values(NoteCategory).includes(overId as NoteCategory)) {
      moveNoteCategory(noteId, overId as NoteCategory)
    }
  }

  return (
    <DndContext sensors={sensors} onDragEnd={handleDragEnd}>
      <div className="kanban-board">
        {columns.map((column) => (
          <div key={column.id} className="kanban-column">
            <div className="column-header">
              <span className="column-title">{column.title}</span>
              <span className="column-count">{column.notes.length}</span>
            </div>

            <SortableContext
              items={column.notes.map((n) => n.id)}
              strategy={verticalListSortingStrategy}
            >
              <div className="column-content">
                {column.notes.map((note) => (
                  <IdeaNoteCard key={note.id} note={note} />
                ))}
              </div>
            </SortableContext>
          </div>
        ))}
      </div>
    </DndContext>
  )
}
```
