## 一、调研背景

织文 WangWen 的「灵感便签」模块用于 AI 自动从对话中提取灵感片段，并支持用户手动创建和管理便签。便签需要支持分类（灵感/伏笔/规则/待办）、关联角色/剧情节点、以及在画布上的自由摆放。核心需求：

- AI 自动从对话中提取灵感
- 便签列表与看板视图
- 便签与角色/剧情节点的关联
- 自由画布摆放（类似便利贴墙）

---

## 二、候选项目分析

### 2.1 React-Kanban —— 看板组件

| 项 | 内容 |
|----|------|
| GitHub | https://github.com/asseinfo/react-kanban |
| Stars | 1.5k+ |
| 技术栈 | React |
| 开源协议 | MIT |

**核心能力**
- 拖拽卡片在不同列之间移动
- 自定义列和卡片渲染
- 支持多板

**适用性**：适合便签的看板视图（按分类分列）

---

### 2.2 React Beautiful DnD —— 拖拽库

| 项 | 内容 |
|----|------|
| GitHub | https://github.com/atlassian/react-beautiful-dnd |
| Stars | 32k+ |
| 技术栈 | React |
| 开源协议 | Apache-2.0 |

**核心能力**
- 列表/网格/看板拖拽排序
- 流畅的拖拽动画
- 支持多列表之间拖拽

**适用性**：若需要自定义看板布局，可用此库自行封装

---

### 2.3 Figma Widget / Excalidraw —— 自由画布

| 项目 | 说明 |
|------|------|
| Excalidraw | 手绘风格白板，支持便签、图形、连线 |
| Figma Widgets | 组件化便签，可自由摆放 |

**适用性**：若需要便签在画布上自由摆放（而非看板列），可参考 Excalidraw 的便签模式

---

## 三、技术选型建议

### 推荐方案：看板视图 + 自由画布双模式

1. **看板模式**：使用 React Beautiful DnD 或自建拖拽逻辑，按分类（灵感/伏笔/规则/待办）分列
2. **自由画布模式**：在无限画布上自由摆放便签，参考 Excalidraw 的便签设计

## 四、设计决策清单

| 决策项 | 方案 |
|--------|------|
| 便签尺寸 | 190px × 自适应高度 |
| 便签样式 | 白底 + 1px 边框 + 顶部分类色条（3px） |
| 结构 | 标题（左）+ 分类 tag + 序号 tag（右） + 内容区 |
| 看板列 | 灵感 / 伏笔 / 规则 / 待办，四列横向排列 |
| 拖拽 | 便签可在列之间拖拽移动 |
| 关联 | 便签可关联角色/剧情节点，显示关联标识 |
