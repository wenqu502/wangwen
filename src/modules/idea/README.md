# 灵感便签模块 (Idea)

负责捕捉和管理创作过程中的碎片化灵感。

## 核心能力

- AI 自动从对话中提取灵感片段
- 手动记录灵感
- 拖拽关联到角色/剧情/体系
- 待处理/已归档状态管理
- 灵感激活（卡文时推荐相关灵感）

## 数据模型

见 `types.ts` 中的 `Idea`。

## AI 交互

AI 监听对话内容，识别潜在灵感片段并询问用户确认。

- `types.ts` — 类型定义
